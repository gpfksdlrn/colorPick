# ColorPick

## 설치

### macOS
1. 위 버튼에서 `.dmg` 다운로드 후 설치
2. 처음 실행 시 "확인되지 않은 개발자" 경고가 뜨면:  
   **Finder에서 앱 우클릭 → 열기 → 열기** 클릭

### Windows
`.exe` 다운로드 후 실행

## 사용법

| 단축키 | 동작 |
|--------|------|
| `⌘ Shift C` | 컬러피커 열기 / 닫기 |
| `Click` | 색상 복사 후 닫기 |
| `F` | 포맷 순환 (HEX → RGB → HSL) |
| `Esc` | 닫기 |

단축키 및 복사 포맷은 트레이 메뉴 → **설정**에서 변경할 수 있습니다.
